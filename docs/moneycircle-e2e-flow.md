# ตัวอย่าง End-to-End ของโปรเจกต์ MoneyCircle

> กรอบการนำเสนอโปรเจกต์: **Problem → Evidence → Solution → Impact → Decision**

---

## Flow Overview

```
┌─────────────┐    ┌─────────────┐    ┌─────────────────────┐    ┌─────────────┐    ┌─────────────┐
│   Problem   │ →  │  Evidence   │ →  │      Solution       │ →  │   Impact    │ →  │  Decision   │
│             │    │             │    │                     │    │             │    │             │
│ ไม่มีวินัย   │    │ Excel/สลิป  │    │ MoneyCircle App     │    │ 30–60 นาที  │    │ ขออนุมัติ    │
│ บันทึกการเงิน│    │ กระจัดกระจาย │    │ + AI Coach + HITL   │    │ → 5–10 นาที │    │ Pilot 1 เดือน│
│ 30–60 นาที/สัปดาห์│ │ ไม่มีภาพรวม │    │ Pilot 15 คน         │    │ วินัย ≥70%  │    │ + manpower  │
└─────────────┘    └─────────────┘    └─────────────────────┘    └─────────────┘    └─────────────┘
```

---

## 1. Problem — ปัญหาธุรกิจหรือกระบวนการ

พนักงานและคนทำงานรุ่นใหม่ **ไม่มีวินัยในการบันทึกและติดตามการเงินส่วนตัว** ทำให้:

- ใช้เวลา **30–60 นาที/สัปดาห์** จดรายรับ–รายจ่ายใน Excel/สมุด แล้วยังสรุปไม่ครบ
- ไม่เห็นภาพรวมว่า **งบประมาณ หนี้สิน และอัตราออม** เป็นอย่างไร
- ขาดแรงจูงใจในการบันทึกต่อเนื่อง → ข้อมูลไม่ครบ → คำแนะนำการเงินใช้ไม่ได้จริง

---

## 2. Evidence — ข้อมูลหรือหลักฐาน

ข้อมูลที่มีอยู่แล้วในองค์กร/ทีม:

- รายการรายรับ–รายจ่ายกระจัดกระจายใน **Excel, สลิป, แอปธนาคาร**
- ไม่มีระบบกลางที่รวม **งบรายหมวด, หนี้สิน, คะแนนสุขภาพการเงิน**
- พนักงานส่วนใหญ่ **ไม่รู้ว่าตัวเองออมได้กี่ % หรือเกินงบหมวดไหน**
- ไม่มีกลไก **gamification / social accountability** ที่ช่วยสร้างวินัย

---

## 3. Solution — workflow / control / Pilot

สร้าง **MoneyCircle** — แอปจัดการการเงินส่วนตัวแบบครบวงจร พร้อม **Human-in-the-Loop (HITL)**:

| ส่วน | รายละเอียด |
|------|------------|
| **แอปหลัก** | Nuxt 3 PWA บน Cloudflare Pages + D1 |
| **ฟีเจอร์หลัก** | บันทึกธุรกรรม, ตั้งงบรายหมวด, จัดการหนี้, คะแนนสุขภาพการเงิน |
| **AI Coach** | Chat ภาษาไทย ดึงข้อมูลจริงจากแอปมาวิเคราะห์ (งบ/หนี้/ออม) |
| **Social** | กลุ่มเพื่อน/ทีม + Leaderboard + Daily Quests |
| **HITL** | ผู้ใช้ยืนยันทุกธุรกรรมและงบเอง — AI แนะนำ ไม่ตัดสินแทน |
| **แนวทางเริ่มต้น** | **Pilot** กับ 1 ทีม/แผนก (15 คน) เป็นเวลา 1 เดือน |

### Tech Stack

- **Frontend:** Nuxt 3, Vue 3, Tailwind CSS, PWA
- **Backend:** Cloudflare Pages Functions, D1 (SQLite)
- **AI:** OpenAI API (Financial Agent)
- **Auth:** Google OAuth + Guest mode, JWT

### Core Features (จากแอปจริง)

| ฟีเจอร์ | คำอธิบาย |
|---------|----------|
| Transaction Tracker | บันทึกรายรับ–รายจ่ายรายวัน |
| Budget Manager | ตั้งเพดานรายหมวด ติดตามการใช้จ่าย |
| Debt Manager | บันทึกหนี้ จำลองแผนปลดหนี้ |
| Financial Score | คะแนน 5 มิติ: งบ, ออม, หนี้, เงินสำรอง, วินัย |
| AI Chat Coach | คำแนะนำการเงินส่วนบุคคลจากข้อมูลจริง |
| Money Circle | กลุ่มเพื่อน/ทีม + Leaderboard + Social Feed |
| Daily Quests | Gamification สร้างวินัยบันทึกต่อเนื่อง |

## 4. Impact — ผลต่อ KPI

| KPI | ก่อน | หลัง (เป้าหมาย Pilot) |
|-----|------|------------------------|
| เวลาสรุปการเงินรายสัปดาห์ | 30–60 นาที | **5–10 นาที** |
| อัตราบันทึกธุรกรรมต่อเนื่อง | ~20% ของพนักงาน | **≥ 70%** (ผ่าน Daily Quest + Streak) |
| การควบคุมงบไม่เกินเพดาน | ไม่มีข้อมูล | ติดตามได้แบบ real-time รายหมวด |
| คะแนนสุขภาพการเงิน | ไม่มี | วัดได้ 5 มิติ (งบ, ออม, หนี้, เงินสำรอง, วินัย) |

**ผลลัพธ์โดยรวม:** พนักงานเห็นภาพการเงินชัดขึ้น ลดความเครียดเรื่องเงิน และมีวินัยบันทึกต่อเนื่อง — ส่งผลต่อ **well-being และ productivity** ของทีม

---

## 5. Decision — สิ่งที่ต้องให้อนุมัติหรือตัดสินใจ

ขออนุมัติ:

1. **Manpower** — ทีมพัฒนา 1–2 คน สำหรับ Pilot และปรับปรุงตาม feedback
2. **Infrastructure** — Cloudflare Pages + D1 (ค่าใช้จ่ายต่ำ ~ฟรี–ต่ำมาก ในช่วง Pilot)
3. **OpenAI API** — สำหรับ AI Financial Coach (ประมาณการตาม usage จริง)
4. **Pilot scope** — ทดลองกับ 1 กลุ่ม/แผนก 15 คน เป็นเวลา 1 เดือน ก่อนขยายทั้งองค์กร

---

## Next Step

หาก Pilot สำเร็จ → **ขยาย MoneyCircle ให้ทุกแผนก** เพื่อลดภาระการจัดการการเงินส่วนตัวแบบ manual และสร้างวัฒนธรรม **financial wellness** ภายในองค์กร

## Appendix: Score Dimensions

คะแนนสุขภาพการเงิน (Financial Score) คำนวณจาก 5 มิติ:

| มิติ | น้ำหนัก | คำอธิบาย |
|------|---------|----------|
| Budget Adherence | 25% | รักษารายจ่ายไม่เกินงบรายหมวด |
| Savings Rate | 20% | อัตราออมเทียบกับรายได้ |
| Debt Health | 25% | วินัยชำระหนี้ตรงเวลา |
| Emergency Fund | 15% | เงินสำรองเทียบกับค่าใช้จ่ายรายเดือน |
| Logging Discipline | 15% | วินัยบันทึกธุรกรรมต่อเนื่อง (streak) |
